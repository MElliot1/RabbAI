# RabbAI 0.5.0

Canonical editable Vite/React application.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Current interaction model

- Tanakh opens initially.
- Tanakh, Rabbinic, and Medieval use a one-open-at-a-time accordion.
- The open group follows the reader's scroll position.
- Synthesis appears at the bottom and has its own left-navigation item.

See the parent folder's `docs/` directory for the project assessment, editorial standards, roadmap, deployment instructions, and changelog.
